# 第二部分：语聊行业名词词典📖

## 产品类型篇

- **蛋包**：软件功能比较少，主要是游戏的一些玩法功能要求比较高，可能没有陪陪营业，都是个人在软件上面玩
- **半市场**：软件功能要求稍多，开黑、相亲、派单可能都要，游戏玩法也要，有引流，有主播正常开厅营业
- **全市场**：对软件功能要求比较高，对客户留存的功能很看重，不要游戏玩法，即便有游戏也是主玩游戏的逻辑，没有抽奖功能，对主播要求比较高，对厅和会长要求比较高，不允许挂麦，引流很多

## 房间与麦位篇

- **1+1+7厅**：最标准的语聊房配置（1个主持麦+1个嘉宾麦+7个标准麦），需明确区分主持麦、标准麦和嘉宾麦的视觉样式
- **主持麦/1麦**：控场、控麦、禁言、拉人上麦，个播厅专属
- **开黑厅**：1+8麦位（1个主持麦+8个标准麦），有开黑队列（可以在开黑队列里申请加入一起打游戏）
- **PK厅**：有厅内PK和跨房PK，厅内分为两组跨房礼物对战
- **上/下麦**：上麦模式分为自由上麦，申请上麦
- **管理操作麦位权限**：情感厅抱上麦、抱下麦、闭麦、锁定麦位、查看资料、踢出房间
- **幽灵麦**：用户离线但麦位显示占用（技术bug）
- **锁麦/锁大头**：把房主锁定在当前麦位状态

## 游戏与玩法篇

- **陪陪**：麦上公会营业人员
- **保底**：多少次砸蛋必中大奖
- **公池**：游戏内的普通奖池，此类型游戏比较基础，一般为抽一次、抽十次、抽三十/抽一百次，简单游戏动效，无大奖动画
- **跳池/高低池**：游戏内不同等级奖池，一般为低中高三个档位奖池
- **彩蛋池**：游戏内有普通奖池和彩蛋奖池，触发彩蛋奖池时
- **累销池**：基于**累计送礼金额**解锁的奖池。用户流水达到特定档位（如50元、500元）可额外抽取一次，常用于刺激大R消费。（*注：此处为业务常见释义补充，如有出入请以产品实际定义为准*）
- **流水**：礼物总收入（厅流水/个人流水），流水统计、排行榜等界面需展示
