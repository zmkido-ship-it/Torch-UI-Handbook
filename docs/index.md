# 火炬技术部 UI 入职手册

亲爱的新员工您好，欢迎加入火炬网络科技有限公司。本手册旨在适配 UI 设计岗位，帮助您快速融入团队、上手工作，规避常见误区。

## 手册结构

本手册分为五个主要部分：

### 第一部分：入门清单🚪

- **设计工具清单** - Figma、Mastergo、iconfont、花瓣
- **协作与交付工具清单** - WPS、Mastergo、企业微信、蓝湖
- **游戏动效设计工具清单** - Photoshop、Adobe Ae、svga导出插件
- **公共账号与权限申请流程清单** - 即梦会员、百度网盘会员、火炬后台管理系统
- **首周闯关任务卡** - 工具安装与账号激活、企业微信绑定、演示包安装、业务黑话词典学习

### 第二部分：语聊行业名词词典📖

- **产品类型篇** - 蛋包、半市场、全市场
- **房间与麦位篇** - 1+1+7厅、主持麦/1麦、开黑厅、PK厅
- **游戏与玩法篇** - 陪陪、保底、公池、跳池/高低池、彩蛋池、累销池、流水

### 第三部分：设计规范速查📋

- **快速入口** - 次级界面规范快速查看、次级界面规范源文件及复用
- **移动端基础尺寸警戒线** - 一级页面尺寸规范、常见新人雷区

### 第四部分：火炬 UI 标准工作流 (从需求到上线)🗺️

- **需求评审** - 接收需求、需求对齐、制定设计计划与工时参考
- **设计执行** - 高保真设计、自查清单、游戏/动效设计
- **交付物标准** - 评审与迭代、交付物要求
- **走查与验收** - 开发对接、走查验收
- **上线后复盘与沉淀** - 经验沉淀、迭代优化

### 第五部分：生存法则与红线🛡️

- **设计评审话术模板** - 有效沟通技巧
- **与前端开发的“互译词典”** - 技术边界理解、文件管理规范
- **职业素养与持续学习** - 产品至上、务实落地、细节控、持续学习

## 使用指南

1. **按照顺序阅读**：建议从第一部分开始，逐步了解公司的工具、术语和流程。
2. **完成闯关任务**：首周任务卡中的任务有助于您快速熟悉工作环境。
3. **参考设计规范**：遇到设计问题时，可随时查阅第三部分的规范速查。
4. **持续更新**：本手册会根据公司发展和业务变化定期更新，请保持关注。

祝您在火炬网络科技有限公司工作顺利，事业有成！
