# 火炬技术部 UI 入职手册

亲爱的新员工您好，欢迎加入火炬网络科技有限公司。本手册旨在适配 UI 设计岗位，帮助您快速融入团队、上手工作，规避常见误区。

## 手册结构

本手册分为三个主要部分：

### &#x20;第一部分：入门前置🚪

- **软硬件工具箱** - 设计工具、协作工具及账号权限
- **业务黑话词典** - 产品类型、房间与麦位、游戏与玩法术语
- **首周闯关任务卡** - 工具熟练与环境搭建、规范与流程熟悉

### &#x20;第二部分：标准工作流🗺️

- **火炬 UI 研发全流程** - 需求评审、设计执行、交付物标准、走查与验收
- **上线后复盘与沉淀** - 经验沉淀、迭代优化

### &#x20;第三部分：生存法则与红线🛡️

- **设计规范速查** - 快速入口、移动端基础尺寸警戒线
- **协作与沟通铁律** - 设计评审话术模板、与前端开发的“互译词典”
- **职业素养与持续学习** - 用户至上、务实落地、细节控、持续学习

## 使用指南

1. **按照顺序阅读**：建议从第一部分开始，逐步了解公司的工具、术语和流程。
2. **完成闯关任务**：首周任务卡中的任务有助于您快速熟悉工作环境。
3. **参考设计规范**：遇到设计问题时，可随时查阅第三部分的规范速查。
4. **持续更新**：本手册会根据公司发展和业务变化定期更新，请保持关注。

祝您在火炬网络科技有限公司工作顺利，事业有成！
